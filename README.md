Movie Mate is a front-end focused React web application designed for discovering, exploring, and managing movies in a simple and interactive way. 
The app allows users to search for movies, view key information through hover interactions, and save their favorite movies to a personalized wishlist. 
It is built as a real-world React project that demonstrates modern UI design, API integration, and state management.

  Application Workflow:
- Users search for movies using the search functionality
- Movie results are fetched asynchronously from an API
- Movie information is displayed dynamically on movie cards
- Additional movie details appear when hovering over a movie
- Users can add movies to their wishlist using the heart icon
- Wishlisted movies are stored and displayed in the Wishlist section

  Tech Stack:
- React (Library)
- Vite (Build Tool)
- JavaScript, HTML, CSS
- JSX (JavaScript XML)

  React Concepts Used:
- Component-Based Architecture
- useState & useEffect Hooks for state and lifecycle management
- Props for data passing between components
- Asynchronous Programming with API Fetch
- Conditional Rendering for dynamic UI updates
- Event Handling (onClick, onChange)
- Error Handling for API responses

Responsive CSS Design for multiple screen sizes
